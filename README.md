# Android-App-for-Digital-Signing-with-RSA-Private-Key-


This Android application is a lab project developed during my Master’s studies.  
It demonstrates how to perform **digital signing using an RSA private key** in an Android environment.

The project was built using **Android Studio** and **Kotlin**.

---

## Overview

The main goal of this application is to show how digital signatures can be used to protect data integrity and authenticity in mobile applications.

With this app, the user can:
- enter or prepare data to be signed
- generate a digital signature using an **RSA private key**
- display the signing result in the application interface

This project is intended for **educational and laboratory purposes** to understand the basics of cryptography and secure mobile development.

---

## Technologies Used

- **Android Studio**
- **Kotlin**
- **RSA Algorithm**
- **Android SDK**

---

## Project Structure

- `MainActivity.kt` → main user interface and app flow
- `DigitalSignatureUtils.kt` → utility class for RSA digital signing logic

---

## How to Run the Project

1. Clone or download this repository
2. Open the project in **Android Studio**
3. Let Gradle sync the project
4. Run the application on an Android emulator or physical device
